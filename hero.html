    Mis servicios

    Lo que hago

    Modelado 3D

    Modelado high y low poly para personajes y assets inorgánicos usando ZBrush, Maya y Blender. Experiencia en esculpido, retopología, texturizado con Substance Painter y creación de animaciones básicas para assets en tiempo real o listos para render.

    Desarrollo Web

    Desarrollo de plataformas sociales, sitios de comercio electrónico y sistemas de reuniones virtuales usando MVC y POO. Dominio de Java, PHP, HTML, CSS, JavaScript y JSON, bajo una estructura cliente-servidor.

    Aplicaciones de Escritorio

    Desarrollo de aplicaciones de escritorio en .Net C# con Visual Studio, enfocadas en usabilidad, rendimiento y soluciones personalizadas a las necesidades de cada usuario.

    Desarrollo de Bases de Datos

    Experiencia con SQL Server y MySQL. Diseño de bases de datos mediante modelos ER y relacionales, normalización, e implementación usando procedimientos almacenados, funciones, vistas y triggers.
